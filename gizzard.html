King Gizzard Performance Scraper

A simple script to scrape the top performances for each song by the band King Gizzard and the Lizard Wizard (as voted on here) and then put them into a Youtube Music playlist.

How does it work?

The script uses a Selenium instance to browse the page. I used selenium as the page itself is dynamic, loading content from a database closed off for public use. This allows the page to load in fully before retrieving each song's top performance.

Once the song name and peformance name are retrieved, the python package YTMusic is used to search Youtube for each. This retrieves a set of songs based on the info searched, which in most cases worked directly with the first result however the script uses some checks to ensure the results are correct. The search results are then used to find the Youtube Music specific ID for each song and add to a playlist.

I don't personally use Youtube Music but unfortunately Apple Music's API is restricted to developer accounts and Deezer's API is off limits. The band has recently took all their music off Spotify as well, so that is also not an option. The script produced great results, with only 5 songs missing and all of which due to youtube musics library being outdated compared to the sites use of Bandcamp as a source.
