King Gizzard Performance Scraper

King Gizzard and the Lizard Wizard have a huge library of songs, and an ever bigger library of live performances which can make things complicated if trying to listen to their best works. Another fan created a community site called Live Gizz Rankings to help deal with this, allowing fans to vote on their favourites and display them on the site. However, this was still not simple enough for myself so I created a simple Python script with the library Selenium to scrape the top performance from each song and add it to a Youtube Music playlist.

How does it work?

The script uses a Selenium instance to browse the page. I used selenium as the page itself is dynamic, loading content from a database closed off for public use. This allows the page to load in fully before retrieving each song's top performance.

Once the song name and performance name are retrieved, the Python package YTMusic is used to search Youtube for each. This retrieves a set of songs based on the info searched, which in most cases worked directly with the first result however the script uses some checks to ensure the results are correct. The search results are then used to find the Youtube Music specific ID for each song and add to a playlist.

I don't personally use Youtube Music but unfortunately Apple Music's API is restricted to developer accounts and Deezer's API is off limits. The band has recently took all their music off Spotify as well, so that is also not an option. The script produced great results, with only 5 songs missing and all of which due to Youtube Music's library being outdated compared to the site which used Bandcamp as it's source.
